Referência de Comandos Linux

Guia rapido para navegar e operar o terminal

Navegacao

pwd
Mostra o caminho do diretorio atual
# Exemplo: /home/root/documentos
ls
Lista arquivos e pastas do diretorio atual
ls -la # lista detalhada com arquivos ocultos
cd [dir]
Muda para o diretorio especificado
cd .. (pai) | cd ~ (home) | cd / (raiz)

Arquivos

cat [arq]
Exibe o conteudo de um arquivo
touch [arq]
Cria arquivo vazio ou atualiza timestamp
mkdir [dir]
Cria um novo diretorio
mkdir -p a/b/c # cria estrutura aninhada
rm [arq]
Remove arquivo
rm -r pasta/ # remove pasta recursivamente
cp src dst
Copia arquivo ou diretorio
mv src dst
Move ou renomeia arquivo

Texto & Busca

echo "txt"
Imprime texto - use > para salvar em arquivo
echo "olá" > arquivo.txt
grep txt arq
Busca padrão em arquivos
grep -r "senha" /etc/ # busca recursiva
find / -name
Encontra arquivos pelo nome
wc -l arq
Conta linhas de um arquivo

Sistema

whoami
Mostra o usuario logado
uname -a
Informações do kernel e sistema
ps aux
Lista processos em execução
top
Monitor de processos em tempo real
df -h
Uso de espaço em disco
chmod +x
Adiciona permissao de execução a um arquivo

Rede (modo simulado)

scan
Simula varredura ARP na sub-rede local
ping [ip]
Testa conectividade com um host
ssh user@ip
Conecta a máquina remota via SSH
crack [ip]
Simula ataque de força bruta SSH (modo educativo)
